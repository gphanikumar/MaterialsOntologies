# Semantic Web Vocabulary

 * [CWA](https://en.wikipedia.org/wiki/Closed-world_assumption) - Closed-World assumption
 * [DL](https://en.wikipedia.org/wiki/Description_logic) - Description Logic
 * [FOL](https://en.wikipedia.org/wiki/First-order_logic) - First Order Logic or first order predicate calculus
 * [IRI](https://www.w3.org/International/articles/idn-and-iri/) - Internationalized Resource Identifier
 * [JSON](https://www.json.org/json-en.html) - JavaScript Object Notation
 * [MDA](https://www.gartner.com/en/information-technology/glossary/model-driven-architecture-mda) - Model-driven Architecture
 * [MDE](https://ict.eu/model-driven-engineering/) - Model-driven engineering
 * [OWA](https://en.wikipedia.org/wiki/Open-world_assumption) - Open-world assumption
 * [RDF](https://en.wikipedia.org/wiki/Resource_Description_Framework) - Resource Description Framework
 * [RDFS](https://www.w3.org/2001/sw/wiki/RDFS) RDF Schema
 * [RDF/XML](https://www.w3.org/TR/rdf-syntax-grammar/) - An XML syntax for RDF in terms of namespaces in XML, the XML Information Set and XML Base.
 * [SAIQL](https://west.uni-koblenz.de/research/software/saiql) Schema And Instance Query Language
 * [SWRL](https://www.w3.org/Submission/SWRL/) - Semantic Web Rule Language
 * [SPARQL](https://en.wikipedia.org/wiki/SPARQL) - SPARQL Protocol and RDF Query Language, [SPARQL 1.1 Query Language](https://www.w3.org/TR/sparql11-query/)
 * [Turtle](https://www.w3.org/TR/turtle/) - Terse RDF Triple Language
 * [URI](https://en.wikipedia.org/wiki/Uniform_Resource_Identifier) - Unified Resource Identifier
 * [XML](https://www.w3.org/XML/) - Extensible Markup Language
